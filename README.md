# Dune Quotes Bot
## Prerequisites
- Node.js
- npm

## Installation
1. Clone the repo or download the code
2. Open ur terminal
3. Install the dependencies:

   ```bash
   npm install
   ```
## Configuration

1. Create .env in the root of ur folder
2. Add ur token & client id (bot's user id), it should look like this:

   ```env
   DISCORD_TOKEN=bot token
   CLIENT_ID=bot id
   ```

## Usage
1. Register the commands by running this:

   ```bash
   node deploy-commands.js
   ```

2. Start bot:

   ```bash
   node index.js
   ```
